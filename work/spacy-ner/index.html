How to deploy locally:

  1. Clone the repository:
$ git clone https://www.github.com/sarbhanub/spacy-ner.git
  1. Preparing a venv:
$ python -m venv .venv && .venv\Scripts\activate
  1. Installing the requirements:
$ pip install -r requirements.txt
  1. Run:
$ python app.py
